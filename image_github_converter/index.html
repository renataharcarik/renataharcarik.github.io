Markdown to HTML Image Converter

When you upload images to Github they are uploaded with their original size. To keep the documentation nice and clean, you can convert the provided github markdown to an html format, where you can set the desired image width. Simply copy the generated html and paste it to your github comment or issue. First option puts the images below each other, each image in one line, option two creates a simple two-column table.
